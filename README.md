## Project Overview

It simulates a real workplace activity where a junior developer is tasked with building a simple Student Readiness Capture System for a company called Kitumaini Digital Solutions.

The system captures student information, validates data, calculates readiness indicators, and displays structured output.

---

## What the Project Does

- Captures student name, age, student number, and grades
- Calculates a readiness status based on the student's grade
- Displays a formatted student profile in the console
- Provides a basic Windows Forms interface for the same functionality

---

## Concepts Demonstrated

- Variables and data types (`string`, `int`, `double`, `bool`)
- Identifiers and keywords
- Arithmetic operators and expressions
- Methods and method parameters
- Method overloading
- Scope (global and local variables)
- Type casting (`int.Parse`, `double.Parse`)
- Console input and output
- Basic Windows Forms interface design

## How to Run the Project

### Console Application
1. Open the solution in **Visual Studio**
2. Set `ConsoleApp` as the startup project
3. Press **F5** or click **Run**
4. Follow the prompts in the console window

### Windows Forms Application
1. Open the solution in **Visual Studio**
2. Set `WindowsFormsApp` as the startup project
3. Press **F5** or click **Run**
4. Fill in the fields and click the **Submit** button
